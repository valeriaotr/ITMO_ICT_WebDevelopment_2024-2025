Kth Permutation Sequence

Level: Hard

Description

The set [1, 2, 3, ..., n] contains a total of n! unique permutations.

By listing and labeling all the permutations in order, we get the following sequence for n = 3:

"123"
"132"
"213"
"231"
"312"
"321"

Given n and k, return the kth permutation sequence.

Example 1:

Input:

n = 3, k = 3

Output:

"213"

Example 2:

Input:

n = 4, k = 9

Output:

"2314"

Example 3:

Input:

n = 3, k = 1

Output:

"123"

Constraints:

  • 1 <= n <= 9
  • 1 <= k <= n!

Solution

Решение использует математический подход для вычисления (k)-й перестановки без генерации всех возможных перестановок.

Основные шаги:

  1. Инициализация:
  2. Создаётся список numbers, содержащий все числа от 1 до n.

  3. Индекс k преобразуется в индекс, основанный на нуле.

  4. Вычисление каждой цифры:

  5. Для каждой позиции вычисляется факториал оставшихся чисел: (fact = (n - 1 - i)!).
  6. Определяется индекс текущего числа как (index = k // fact).
  7. Добавляется выбранное число в результат, а затем удаляется из списка numbers.

  8. Обновляется значение (k = k \% fact).

  9. Возврат результата:

  10. Объединяются все выбранные цифры для формирования строки, представляющей перестановку.

Примечания:

  • Алгоритм имеет временную сложность (O(n^2)), так как удаление элементов из списка занимает (O(n)).
  • Используется (O(n)) дополнительной памяти для хранения списка numbers.
