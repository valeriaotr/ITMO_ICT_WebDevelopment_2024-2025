Лабораторная №4: Медицинская система управления

Основные возможности

  • Управление записями о приемах:
  • Получение списка приемов.
  • Добавление нового приема.
  • Удаление существующего приема.

  • Обновление данных приема.

  • Управление врачами:

  • Получение списка врачей.
  • Добавление нового врача.

  • Удаление врача.

  • Управление медицинскими картами:

  • Создание новой медицинской карты.
  • Получение списка медицинских карт.

  • Удаление медицинской карты вместе с пациентом.

  • Управление пациентами:

  • Получение списка пациентов.
  • Добавление нового пациента.
  • Обновление данных пациента.

  • Удаление пациента.

  • Управление медицинскими услугами:

  • Получение списка доступных услуг.
  • Добавление новой услуги.
  • Обновление данных услуги.
  • Удаление услуги.

Основные файлы и их назначение

axiosInstance.js

  • Настраивает экземпляр Axios с указанием базового URL и токена авторизации.

api/appointments.js

  • Функции для работы с приемами:
  • getAppointments
  • addAppointment
  • deleteAppointment
  • updateAppointment

api/doctors.js

  • Функции для работы с данными врачей:
  • getDoctors
  • addDoctor
  • deleteDoctor

api/medicalCards.js

  • Функции для работы с медицинскими картами:
  • createMedicalCard
  • getMedicalCards
  • deleteMedicalCard

api/patients.js

  • Функции для работы с пациентами:
  • getPatients
  • createPatient
  • updatePatient
  • deletePatient

api/services.js

  • Функции для работы с медицинскими услугами:
  • getServices
  • createService
  • updateService
  • deleteService

Дополнительные функции

  • getCabinets: Получение списка кабинетов.
